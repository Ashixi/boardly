Роадмап проекту

Наш план розвитку та майбутні можливості

Зараз: Фундамент

Реалізовано систему локальних дошок, P2P-хостинг на базі WebRTC, керування файловою структурою та систему безпечної авторизації через Email.

Етап 1: Інтеграція GitHub

Синхронізація ваших дошок з репозиторіями. Можливість автоматичного збереження змін у Git та версіонування контенту безпосередньо з інтерфейсу Boardly.

Етап 2: Повна підтримка .md

Впровадження потужного редактора Markdown-файлів. Створення документації, нотаток та вікі-сторінок прямо на ваших дошках з підтримкою Rich Text та прев'ю.

Етап 3: Хмарна екосистема

Інтеграція з популярними хмарними сховищами (Google Drive, Dropbox). Можливість працювати з вашими дошками з будь-якого пристрою, зберігаючи контроль над даними.

Етап 4: Інтелектуальний помічник (AI)

Впровадження штучного інтелекту для аналізу контенту дошок, автоматичної організації завдань та генерації ідей у реальному часі під час командних сесій.
