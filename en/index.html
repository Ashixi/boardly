Development Roadmap

Phase 1: Core Foundation

Infinite canvas, basic tools (pen, shapes, text), and local file system saving. Windows & Linux support.

Phase 2: P2P Connectivity

WebRTC integration for real-time multiplayer. Guest join via ID. Basic pointer synchronization.

Phase 3: Advanced Collaboration (In Progress)

Voice chat integration, voting system, and advanced permissions for board owners.

Phase 4: Cloud Optionality

Optional encrypted cloud backup for users who want persistence across devices without P2P.
